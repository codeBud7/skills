# Skills

Skill collection and workflow, to stay in loop as human.

**Order:** Seed (plan) ‚Üí Harvest (ship) ‚Üí Cultivate (deslop).

- **Seed** ‚Äî plan before code (Cursor, Claude Code, or `plans/*.md`)
- **Harvest** ‚Äî `repo-safety` ‚Üí `tdd-cycle` ‚Üí `local-quality-gate` ‚Üí `cultivate` ‚Üí `docs-sync` ‚Üí `draft-pr` ‚Üí `ci-green`; can consume the Seed plan file and tick todos during ship (plan-driven Build)
- **Cultivate** ‚Äî lean AI-written code, same behavior; runs as harvest's post-impl deslop step (re-verify green) and standalone

**Subagents:** read-only discovery or independent parallel work ‚Üí delegate subagent(s); main thread keeps decisions and edits.
