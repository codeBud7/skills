# 🚜 Skills
Personal collection of handcrafted skills to still be the human in the loop.

The core idea of the skills is following a strict order, using farming terminology:

1. Seed 🌱
2. Harvest 🚜
3. Cultivate 🌿

## Seed
Human in the loop planning what to harvest.

## Harvest
Getting things done! 

Consists of...
...repo-safety
...TDD
...local-quality-gate
...draft-pr
...ci-green

## Cultivate
